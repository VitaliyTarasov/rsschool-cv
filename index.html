Array Deep Count

Array.prototype.length will give you the number of top-level elements in an array. Your task is to create a function deepCount that returns the number of ALL elements within an array, including any within inner-level arrays.


    function deepCount(a){
      return a.reduce((i,  j) => {
        return i + (Array.isArray(j) ? deepCount(j) : 0) },  a.length);
    }

    deepCount([1, 2, [3, 4, [5]]]);  //  7
